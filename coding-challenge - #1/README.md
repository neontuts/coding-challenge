# Coding Challenge #1 - SPY Number

## Problem

**Write a program to check if number is a SPY Number or Not.**

```
A Number is called SPY Number if the sum and product of it's digits are equal.

For Ex : 123 is a SPY Number
	(1+2+3) = (1*2*3)
```

## Test Cases


| INPUT |    OUTPUT    |
|-------|--------------|
| 1124  | SPY          |
| 1254  | NOT SPY      |
| "123" | NOT A NUMBER |
| 321   | SPY          |
| 2020  | NOT SPY      |
